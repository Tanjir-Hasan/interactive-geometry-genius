Q&A about JavaScript

Question: 1

What is DOM and what is the purpose of DOM?

Answer:

Definition: The full form of the DOM is "Document Object Model". It's an Application Programming Interface which is Cross Platform and Language Independent. It modifies documents structure, style, content and represent HTML, XHTML, XML Document as a tree.


Purpose of DOM:


  1. DOM provides the access of a document like HTML, XHTML and XML.
  2. DOM makes a platform to be language neutral.
  3. DOM provides the dynamic usage of the script.
  4. DOM provides a way to update the document’s content, structure and style.
  5. DOM is used with the language and used to create the objects that can be used as a model to allow the interface to be developed easily.

Question: 3

What is event bubble?

Answer:

Definition: When an element receives an event, and that event is transmitted to its parent elements in the DOM tree until it gets to the root element. This concept is called event bubble.


Question: 4

What is a callback function and why will you use it?

Answer:

When a function is sent as an argument then it executes after the main function has been executed it’s called a Callback Function. The callback function is very useful for its efficiency. Callback Functions give a way to you to set up your program in such a way that events will trigger the correct code. It’s very common in programs with GUIs where users can click on any element on the UI and the appropriate code will be run.
